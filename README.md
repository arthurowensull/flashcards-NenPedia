# NenPedia — Enciclopédia de Nen

Este projeto é uma página web feita com HTML e CSS, inspirada no universo de Hunter x Hunter.

## Tema

O site apresenta informações sobre o sistema de poder chamado Nen, explicando conceitos importantes como aura, tipos de Nen, técnicas básicas e avançadas, usuários conhecidos e criação de habilidades.

## Como funciona

A página começa com uma tela de entrada em formato de livro. Ao clicar no livro, o conteúdo do site aparece.

Depois disso, o visitante pode navegar pelas seções e passar o mouse sobre os flashcards para ver explicações mais completas sobre cada assunto.

## Conteúdo do site

* Introdução ao universo de Hunter x Hunter
* Explicação sobre o Nen
* Flashcards sobre fundamentos do Nen
* Tipos de Nen
* Técnicas de Nen
* Usuários conhecidos
* Guia para criar uma habilidade de Nen equilibrada

## Tecnologias usadas

* HTML
* CSS

## Objetivo

O objetivo do projeto é praticar HTML e CSS criando uma página interativa, organizada e informativa sobre o sistema de Nen de Hunter x Hunter.
